Embed your posts as a list or photo grid widget from [Findery](https://findery.com/ "Share the places you love and explore the world.")

== Description ==

Findery 

Findery is all about places.

From local lore to travel stories, share the places you love, learn more about your favorite places and the places youâ€™ve always wanted to visit.

Usage:

*   Go to Appearance > Widgets.
*   Drag "Findery Notes" to the sidebar or other area where you want it to display.
*   Change the Findery username to your username. If you don't already have an account, go to [Findery](https://findery.com/ "Share the places you love and explore the world.") to 
register.
*   [Optional] Change the embed layout as either List or Photo Grid.
*   [Optional] Change the width and height if desired.
*   [Optional] Add custom CSS styles. (Default: None)
*   [Optional] Show a frame border. (Default: Off)
*   [Optional] Display scroll bars. (Default: On)
*   Click "Save"

== Installation ==

1. Upload `findery_notes_widget.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. See instructions above for how to use the widget.
